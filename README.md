# 0. _**0.1.0 -> 1.0.0**_ 변경점
- 사용자 사용법 변경.
- 내부 로직 변경.
- 16진수 기준, 0부터 F까지 맞출 숫자의 범위를 지정할 수 있음.
- 위에서 입력한 숫자 범위를 초과하지 않는 선에서, 맞출 숫자의 갯수를 지정할 수 있음.
- 0.1.0 에서 변경되었던 위 로직을 사용자 입력단에도 추가함.
